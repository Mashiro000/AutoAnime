# 🎬 AutoMedia AI：让乱码文件名重获新生

> **“下完资源发现全是乱码？强迫症表示不能忍！”**
>
> 这是一款懂你的全能影音整理工具。它不只是简单的重命名，而是利用 **DeepSeek / Gemini / 豆包** 等大模型的“大脑”，配合 **TMDB** 的官方数据，帮你把乱七八糟的下载文件夹变成精美的影视库。

---

### ✨ 它能为你做什么？

* **🧠 智能脑补**：不管文件名是 `[LoliHouse] Kaguya-sama...` 还是 `阿凡达.Way.of.Water.2022`，AI 都能一眼认出它是谁。
* **📂 自动归档**：动漫归动漫，电影归电影。它会乖乖建立类似 `动漫/番剧名/Season 1/` 这样的标准目录。
* **🏷️ 官方标准**：接入 TMDB API 后，文件名会自动校正为官方中文译名，让 Plex、Emby、Jellyfin 刮削再也不报错。
* **🔗 硬链接魔法**：在整理的同时不占用双倍空间，原始文件还能继续保种，简直是 PT 党的福音。

### 🚀 快速上手

1.  **填入 Key**：选一个你喜欢的 AI 服务商（推荐硅基流动，响应飞快）。
2.  **选好路径**：告诉它哪里是下载区，哪里是你的影库。
3.  **看进度条**：顶部的“进度胶囊”亮起后，点击启动，剩下的交给 AI。
